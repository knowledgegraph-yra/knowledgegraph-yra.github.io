ナレッジグラフ若手の会

Follow @kg_wakate


ナレッジグラフ若手の会は、ナレッジグラフ関連の研究者の交流・情報交換の場です。

論文読み会やLT会などのイベント[connpass] や情報交換を行い、様々なドメインの研究者による「ナレッジグラフコミュニティ」の活性化を図っています。

年齢、組織、職種、在学の有無に関係なく、ナレッジグラフ関連分野に興味を持ち、若手の心を持った方のご参加を歓迎いたします。

イベント

ナレッジグラフ若手の会では、様々なイベントを企画しています。現在開催している主なイベントの種類は以下のとおりです。

  • 論文読み会　ISWCやESWCなどのトップカンファレンスをはじめとする、ナレッジグラフに関連する論文を持ち寄って紹介しあうイベントです。
  • LT会　分野や経験を問わず発表者を募集し、ナレッジグラフに関する知識と視点を5～10分ずつシェアしあうイベントです。
  • 講演会　ナレッジグラフ関連分野で第一線で活躍されている方に、ナレッジグラフのトレンドや社会での使われ方、これからの広がりについてお話しいただくイベントです。20分程度の講演と、2～30分程度の質疑応答タイムで構成されます。

    • ナレッジグラフ若手の会の概要

    目的

    • 国内に分散しているナレッジグラフ (KG) 系研究者の交流の場をつくること
    • KG系研究コミュニティを活性化すること


    設立の経緯

    近年、ナレッジグラフの機械学習応用や深層学習と知識の融合などをきっかけとして、

    これまでセマンティックWebやオントロジーを対象としていなかった研究者にも、そのような技術が注目されるようになってきました。

    その結果、それぞれのドメインで独自に頑張るナレッジグラフ研究者が増え、研究コミュニティが分散するようになってきました。

    その状況を受けて「ナレッジグラフ研究者が集まる場所をつくろう、コミュニティを活性化させよう」という動きが大きくなり、

    若手研究者を中心に「ナレッジグラフ若手の会」が設立されました。
